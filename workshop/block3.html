Was ist ein Prompt — und warum ist er wichtig?
Folie 2 / 11
Ein Prompt ist...
  • 💬Die Eingabe, die Sie Claude schicken
  • 📝Eine Frage, ein Auftrag oder eine Beschreibung
  • 🎯Der wichtigste Hebel für gute Ergebnisse
  • ⭐Je klarer der Prompt, desto besser die Antwort

Wie ein neuer Mitarbeiter

Stellen Sie sich vor, Sie haben ab heute eine neue, sehr intelligente Assistentin. Sie kennt Ihr Unternehmen noch nicht, hat keinen Kontext und kann nur das umsetzen, was Sie ihr sagen. Je genauer Ihre Anweisung, desto besser das Ergebnis.

Claude ist kein Gedankenleser. Was Sie nicht sagen, muss er raten — und das geht oft schief.

❌ Was passiert bei einem schlechten Prompt?
Folie 3 / 11
Vager Prompt

"Schreib eine E-Mail"

Claudes Problem

Claude weiß nicht: An wen? Über was? Welcher Ton? Wie lang? Formell oder locker? → Ergebnis: Eine generische E-Mail, die Sie komplett umschreiben müssen.

Kein Empfänger

"E-Mail schreiben" — wer bekommt sie? Kunde, Lieferant, Kollege?

Kein Thema

Was soll die E-Mail sagen? Was ist der Anlass?

Kein Ton

Formell? Freundlich? Dringend? Kurz oder ausführlich?

Ein schlechter Prompt kostet mehr Zeit als er spart — man schreibt am Ende selbst.

👤 Zutat 1: Rolle — wer soll Claude sein?
Folie 6 / 11
Was eine Rolle bewirkt
  • 🎭Claude passt Ton, Fachtiefe und Stil an die Rolle an
  • ⚖️Ein "Rechtsanwalt" schreibt anders als ein "freundlicher Assistent"
  • 🔧Rollen müssen nicht realistisch sein — sie steuern die Ausgabe
Beispiele
  1. 1"Als erfahrener Geschäftskorrespondenz-Experte..." → Formelle, präzise Briefe
  2. 2"Als hilfsbereite Kollegin..." → Locker, direkt, unkompliziert
  3. 3"Als Übersetzer für Baufachsprache..." → Präzise Fachterminologie
  4. 4"Du bist meine persönliche Assistentin..." → Angepasst an Ihre Bedürfnisse
Optional — viele Prompts funktionieren auch ohne explizite Rolle. Hilfreich bei spezifischen Stilen oder Fachgebieten.

📋 Zutat 2: Kontext — die Situation erklären
Folie 7 / 11
Was zum Kontext gehört
  • 🏢Wer sind Sie / Ihr Unternehmen?
  • 👤An wen richtet sich der Text?
  • 📜Was ist die Vorgeschichte?
  • ℹ️Welche Informationen sind wichtig?
Prompt-Vergleich
Ohne Kontext
"Schreib eine Mahnung."
Mit Kontext
"Schreib eine Mahnung für die Firma Stahlbau Krenn KG. Rechnung Nr. 2025-112 über € 4.800,– ist seit 45 Tagen offen. Wir hatten bereits eine freundliche Erinnerung geschickt. Diesmal soll die Mahnung bestimmter sein, aber noch nicht mit rechtlichen Schritten drohen."
Kontext ist das Wichtigste — mehr Kontext bedeutet fast immer ein besseres Ergebnis.

🎯 Zutat 3: Ziel — was soll herauskommen?
Folie 8 / 11
Das Ziel klar benennen
  • ✉️"Schreib eine E-Mail" vs. "Erstelle einen Entwurf für eine E-Mail"
  • 📝"Zusammenfassung" vs. "5 Stichpunkte, die wichtigsten Fakten"
  • 📊"Hilf mir" vs. "Erstelle eine Tabelle mit 3 Spalten"
Häufige Zielformulierungen
  1. 1Schreib / Erstelle / Formuliere: Für neue Texte und Dokumente
  2. 2Fass zusammen / Kürze: Für Zusammenfassungen und Kürzungen
  3. 3Übersetze / Formuliere um: Für Spracharbeit
  4. 4Erkläre / Beantworte: Für Fragen und Erklärungen
  5. 5Erstelle eine Tabelle / Liste: Für strukturierte Ausgaben
Das Ziel sagt Claude: Was ist fertig, wenn er die Antwort schickt?

📐 Zutat 4: Format — wie soll es aussehen?
Folie 9 / 11
Format-Angaben
  • 📏Länge: "Max. 3 Sätze" / "Ausführlich mit Begründung"
  • 🎨Ton: "Formal" / "Freundlich" / "Sachlich" / "Dringend"
  • 🗂️Struktur: "Als Liste" / "Als Fließtext" / "Mit Überschriften"
  • 🌐Sprache: "Auf Deutsch" / "Österreichisches Deutsch"
  • 💾Ausgabe: "Als Word-Datei" / "Als Tabelle" / "Mit Briefkopf"
Praktisches Beispiel
"Fass diesen Vertrag in maximal 5 Stichpunkten zusammen. Fokus auf: Fristen, Zahlungsbedingungen und Haftung. Sprache: einfaches Deutsch, keine Juristensprache."
Format ist optional — aber es verhindert, dass Claude einen Roman schreibt, wenn Sie nur 2 Sätze brauchen.

🔄 Nachfassen — das Gespräch weiterführen
Folie 10 / 11
Claude merkt sich den Chat-Kontext
  • 💾Alles im selben Chat ist Claude bekannt
  • 🔄Kein erneutes Erklären nötig
  • ✏️Einfach sagen, was besser werden soll
Typische Nachfass-Prompts
  1. 1"Kürzer bitte" — auf 3 Sätze kürzen
  2. 2"Formeller" — Ton anpassen
  3. 3"Füge eine Frist hinzu: bis 30. Mai" — Detail ergänzen
  4. 4"Mach daraus eine Liste" — Format ändern
  5. 5"Das war gut, jetzt dasselbe für Lieferant Maier" — Variante erstellen
Claude ist kein Automat — er ist ein Gesprächspartner. Zwischenergebnisse verfeinern ist normal und gewollt.

📋 Ihr Spickzettel — die wichtigsten Regeln
Folie 11 / 11
Die 5 goldenen Regeln
  • ✓Konkret sein: Namen, Daten, Zahlen immer angeben
  • ✓Ziel benennen: Was soll am Ende herauskommen?
  • ✓Ton angeben: Formell, freundlich, kurz, ausführlich
  • ✓Nachfassen: Erstes Ergebnis ist ein Entwurf, nicht das Finale
  • ✓Kontext geben: Je mehr Claude weiß, desto besser
Der universelle Prompt-Baukasten
[Rolle, optional] + [Kontext] + [Ziel] + [Format]

Beispiel ausgefüllt:

"[Als Büroprofi bei einem Bauunternehmen] schreib [eine Auftragsbestätigung für Kunde Stadtgemeinde Linz, Pflasterarbeiten, € 38.500,–] [als professionellen Brief] [mit Briefkopf Grundstein Bau GmbH, max. 1 Seite]."

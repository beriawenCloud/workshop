💬 Der Chat — wo alles beginnt
Folie 3 / 12
Was der Chat kann
  • ✓Fragen stellen und Antworten erhalten
  • ✓Texte schreiben, umformulieren, übersetzen
  • ✓Dateien hochladen und analysieren (PDF, Word, Bilder)
  • ✓Code und Tabellen erstellen
  • ✓Websuche (Such-Icon im Eingabefeld aktivieren)
Was der Chat nicht kann
  • ✗Keinen Kontext aus früheren Chats merken (ohne Projekte)
  • ✗Keine Dateien auf Ihrem Computer speichern (ohne Cowork)
  • ✗Nicht eigenständig E-Mails versenden
Jeder neue Chat beginnt bei null — was Claude aus einem alten Chat wissen soll, müssen Sie neu erklären oder in einem Projekt speichern.

📁 Projekte — Claude mit Gedächtnis
Folie 4 / 12
Was ein Projekt speichert
  1. 1Anweisungen: Briefkopf, Ton, Unternehmensinfos, wiederkehrende Formulierungen
  2. 2Dateien: Vorlagen, Checklisten, Handbücher — Claude kennt den Inhalt
  3. 3Kontext: Alles bleibt erhalten, jeder neue Chat im Projekt startet mit diesem Wissen
Wann ein Projekt sinnvoll ist
  • ✉️Schriftverkehr: Immer gleicher Briefkopf und Ton
  • 📝Protokolle: Immer gleiche Struktur und Teilnehmer
  • ✅Checklisten: Wiederkehrende Standards
  • 👥Team: Mehrere Personen nutzen dieselben Vorgaben
Einmal einrichten, dauerhaft schneller arbeiten.

📄 Artifacts — Dokumente direkt im Chat
Folie 5 / 12
Was ist ein Artifact?

Wenn Claude ein längeres Dokument, eine Tabelle oder eine strukturierte Ausgabe erstellt, erscheint diese in einem eigenen Bereich rechts neben dem Chat — dem Artifact-Bereich. Dort lässt sich der Inhalt direkt kopieren.

  • 📄Formatierte Texte und Briefe
  • 📊Tabellen (direkt nach Excel kopieren)
  • 💻HTML-Seiten und Code
  • 📋Strukturierte Listen und Protokolle
Was Artifacts NICHT sind
Artifacts sind keine gespeicherten Dateien! Der Inhalt existiert nur im aktuellen Chat. Zum Speichern: Inhalt kopieren und in Word/Excel einfügen — oder Claude bitten, eine Datei zu erstellen (mit Cowork).

⚙️ Personalisierung — Claude kennenlernen
Folie 6 / 12
Wo finden Sie die Einstellungen

Im Browser: oben rechts → Ihr Profilbild → Einstellungen. In der Desktop App: unten links → Ihr Profilbild → Einstellungen. Dort können Sie Claude mitteilen, wer Sie sind und wie er mit Ihnen kommunizieren soll.

  1. 1Profil: Name, Beruf, was Claude über Sie wissen soll
  2. 2Kommunikation: Wie ausführlich, welcher Ton, welche Sprache
  3. 3Speichern: Einstellungen gelten für alle neuen Chats
Beispiel-Einstellung
"Ich bin Sekretärin bei einem Bauunternehmen in Österreich. Ich nutze Claude hauptsächlich für Schriftverkehr, Protokolle und Dokumentenerstellung. Bitte schreib immer auf Deutsch, professionell und klar. Keine unnötig langen Antworten."
Diese Einstellungen sparen Ihnen bei jedem Chat das erneute Erklären — Claude weiß dann immer, wer Sie sind.

💻 Claude Code — für Entwickler (zur Info)
Folie 8 / 12
Was ist Claude Code?

Claude Code ist ein separates Kommandozeilen-Tool für Software-Entwickler. Es läuft im Terminal und ermöglicht es, Code direkt in Projekten zu schreiben und auszuführen. Das ist NICHT derselbe "Code"-Tab, den man manchmal im Chat sieht.

Für den Büroalltag nicht relevant — Claude Code ist ein spezialisiertes Entwickler-Tool. Wir erwähnen es nur, damit Sie es nicht verwechseln, falls Sie den Begriff hören.
Was Sie stattdessen nutzen
  • 🌐Claude.ai im Browser: Für alle täglichen Aufgaben
  • 🖥️Claude Desktop App (Cowork): Für Dateioperationen
  • 🔌Konnektoren: Für Verbindung zu Google/Microsoft etc.

🧠 Memory — Claude merkt sich wichtige Dinge
Folie 10 / 12
Was ist Memory?

Memory ist eine optionale Funktion: Claude kann sich wichtige Informationen über Sie dauerhaft merken — über einzelne Chats hinaus. Zum Beispiel: Ihr Name, Ihr Unternehmen, Ihre Präferenzen.

  1. 1Memory aktivieren: Einstellungen (Desktop App: unten links | Browser: oben rechts) → Memory → einschalten
  2. 2Claude speichert automatisch: Was er für relevant hält
  3. 3Sie können prüfen und löschen: Einstellungen → Memory → anzeigen
Unterschied Projekt vs. Memory
ProjektMemory
Was?Anweisungen + DateienFakten über Sie
Wo?Ein ProjektAlle Chats
Kontrolle?Sie schreiben esClaude lernt es
Löschen?ProjektanweisungenEinstellungen → Memory
Memory muss erst aktiviert werden — es ist standardmäßig ausgeschaltet.

🔒 Einstellungen — Datenschutz und Unternehmenskonto
Folie 11 / 12
Datenschutz-Einstellungen
  • 🛡️Trainings-Opt-out: Im Pro-Plan werden Ihre Chats standardmäßig nicht für Training verwendet
  • 🗑️Daten-Löschung: Chats können jederzeit gelöscht werden
  • 🚫Kein Speichern sensibler Daten: Keine echten Kundendaten, Passwörter oder vertrauliche Dokumente eingeben

Microsoft 365 Integration

Ob Claude auf Outlook, OneDrive oder Teams zugreifen darf, hängt von Ihrer IT-Abteilung ab. In vielen Unternehmen müssen solche Verbindungen erst freigegeben werden. Fragen Sie ggf. kurz nach, bevor Sie Konnektoren einrichten.

Grundregel: Nichts in Claude eingeben, was Sie nicht auch per E-Mail verschicken würden.

Was sind Projekt-Skills?

In einem Projekt können Sie nicht nur Anweisungen schreiben, sondern auch ganze Dokumente hochladen: Preislisten, Vertragsvorlagen, interne Richtlinien. Claude kennt dann deren Inhalt und kann darauf zugreifen.

Beispiel: "Welche Zahlungsbedingungen stehen in unserem Standardvertrag?" → Claude liest das hochgeladene Dokument und antwortet.

In Block 4 lernen Sie

  • 📐Wie Sie ein Projekt richtig einrichten
  • 📎Welche Dateien sinnvoll sind
  • ✏️Wie Sie Anweisungen formulieren, die Claude wirklich helfen
  • 🎯Praxis-Demo: Projekt für Schriftverkehr anlegen
Projekte + hochgeladene Dokumente = Claude als persönlicher Assistent, der Ihr Unternehmen kennt.
